# **cbrfree-wifi-network-analysis**
Analyzed public Wi-Fi usage data from the CBRfree network in Canberra to identify traffic patterns, peak usage periods, and access point performance. Built a Tableau dashboard to visualize sessions, throughput, and total network traffic, providing insights for network optimization and infrastructure planning.
